# react-thread-pool

This is an multiprocessing library based on react/event-loop.
Status: alpha
Tests: none

## Requirements
- Linux/Unix platform
- PHP >= 5.4
- react/event-loop
- evenement/evenement
- [optional] Libevent (pect/libevent)
